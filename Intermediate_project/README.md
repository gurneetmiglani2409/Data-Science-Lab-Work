# Loan Approval Prediction

## Description
This project aims to predict whether a loan will be approved or not using machine learning techniques.

## Dataset
- Source: Kaggle / Provided dataset
- Description: The dataset contains information like income, loan amount, credit history, etc.

## Steps Performed
1. Data Cleaning (handling missing values)
2. Exploratory Data Analysis
3. Data Visualization
4. Model Building using classification algorithm

## Results
- The model was able to predict loan approval
- Important factors affecting approval were identified

## Tools Used
- Python
- NumPy, Pandas
- Matplotlib / Seaborn
- Scikit-learn

## Conclusion
The project shows how machine learning can be used to predict loan approval and understand important features.
