# Customer Segmentation by K-means Clustering Algorithm

## Description
This project focuses on segmenting customers into different groups based on their behavior using clustering techniques.

## Dataset
- Source: Kaggle / Mall Customers dataset
- Description: The dataset includes customer information like age, income, and spending score.

## Steps Performed
1. Data Cleaning
2. Exploratory Data Analysis
3. Data Visualization
4. Model Building using KMeans clustering

## Results
- Customers were divided into different clusters
- Each group showed different spending behavior

## Tools Used
- Python
- NumPy, Pandas
- Matplotlib / Seaborn
- Scikit-learn

## Conclusion
This project helps in understanding customer behavior and grouping them effectively.
